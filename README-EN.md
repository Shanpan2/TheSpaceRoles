## Chat Commands
| Command                                     | Function                                          |
| ------------------------------------------- | ------------------------------------------------- |
| /help<br>/h<br>/?                           | Displays a list of chat commands and key commands |
| /ver<br>/v                                  | Displays the version of the TSR that is installed |
| /name `<string>`<br>/n `<string>`               | Change your own name.                             |
| /gamecommand<br>/ge                         | Can force the game to end. There are also key commands to end game                         |
| /help<br>/h                                 | Show command description                          |
| /namecolor<br>/nc `<colorcode>`              | Change the color of your name                     |
| /lobytimer <true/false><br>/It <true/false><br>  | `true = Enabled`<br>`false = Disable`<br>Lobby timer can be turned on and off with "true" and "false".<br>For more information about the lobby timer.(Notpreparetion)                         |

## Key Commands
| HotKey      | Function                                                                           | Usable Scene  |
| ----------- | ---------------------------------------------------------------------------------- | ------------- |
| `Left Shift`+`H`+`Right Shift` | Force End Game                                            | In Game       |
| `Left Shift`+`S`+`Right Shift` | Skip meeting to end                                       | In Game       |
| `C`                            | Cancel game start                                         | In Countdown  |
| `V`                            | Start the game immediately                                | In Countdown  |
| `Ctrl`+`Z`                     | Undo (chat input)                                         | Chat          |
| `Ctrl`+`Y`                     | Redo (chat input) (`Ctrl+Z` to undo what you have undone) | Chat          |
| `↑`                            | Paste the above chat into the chat input field            | Chat          |
| `↓`                            | Paste the chat below into the chat input field            | Chat          |
| `Ctrl`+`C`                     | Copy the text                                             | Chat          |
| `Ctrl`+`V`                     | Paste the text                                            | Chat          |
| `Ctrl`+`X`                     | Cut the text                                              | Chat          |

## Impostors<br>
## Vampire<br>
#### Description(intro)<br>
I'm a vampire.
#### Role Description<br>
Vampires can make delayed kills.<br>
 When they bite a player, the bitten player dies after a set number of seconds. <br>
However, near garlic, which all players can set up, they cannot bite and it is a normal kill. <br>
Use delayed kills to confuse the information.
#### Setting
## SerialKiller<br>
#### Description(intro)<br>
Uncontrollable urge to kill.<br>
#### Role Description<br>
Serialkiller has a very short kill cooldown, but if they do not make a kill for a certain amount of time, they will not be able to control their murderous impulses and will kill themselves...<br>
Once you make a kill, a timer starts until you kill yourself, and the suicide timer resets depending on whether you make a kill, or the meeting starts. Just keep on killing. There is no time to waste.
#### Setting
